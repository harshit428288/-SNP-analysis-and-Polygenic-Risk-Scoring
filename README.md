# 🧬 Interactive Explainable Genomic Risk Visualization System

An interactive genomic analysis dashboard that predicts disease risk using SNP analysis and Polygenic Risk Scoring (PRS). The system dynamically visualizes SNP contribution percentages using explainable AI concepts and interactive graphs.

---

# 🚀 Features

- SNP-based disease risk prediction
- Polygenic Risk Score (PRS) calculation
- Interactive SNP sliders
- Disease-specific SNP mapping
- Explainable SNP contribution analysis
- Dynamic pie charts and bar graphs
- Streamlit-based dashboard
- Bioinformatics and genomics visualization

---

# 🧠 Diseases Included

- Diabetes
- Heart Disease
- Obesity
- Breast Cancer
